La primera publicación que se refiere a este problema como el "problema del viajante
de comercio" se encuentra en el informe de 1949 de la RAND Corporation por 
Julia Robinson, "On the Hamiltonian game (a traveling salesman problem)".

Así es como define el problema

"El propósito de esta nota es dar un método para resolver un problema relacionado
con el problema del viajante de comercio. Parece que vale la pena dar una
descripción del problema original. Una formulación es encontrar la ruta más corta
para un vendedor de que parte de Washington, visita todas las capitales de los
estados y luego regresa a Washington.

Más en general, encontrar la CURVA CERRADA más corta que contenga n puntos dados 
en el plano".

por ejemplo con el siguiente conjunto de ciudades:
0, 0
1, 0
2, 0
0, 1
1, 1
2, 1
1, 2
2, 2
que puede presentarse de la siguiente manera:
+ + +
+ + +
  + +
el camino más corto es:
 _____
|__   |
   |__|

por lo que debe imprimir la longitud de este camino que es:
8.00

Escribe un programa que lea un conjunto de coordenadas de ciudades de la forma 
'%f, %f\n' de la entrada estándar e imprima la longitud del camino
más corto posible que contenga todas estas ciudades bajo la forma '%.2f'.

Su programa no será probado con más de 11 ciudades.

Encontrarás en este directorio un archivo tsp.c que contiene todas las partes
aburridas de este ejercicio y archivos de ejemplo para ayudarte a probar tu programa.

pista: para usar sqrtf, añade -lm al final de tu comando de compilación.

Por ejemplo esto debería funcionar:
$> cat square.txt
1, 1
0, 1
1, 0
0, 0
$> ./tsp < square.txt | cat -e
4.00$

The first publication referring to this problem as the "traveling salesman 
problem" is found in the 1949 RAND Corporation report by Julia Robinson, 
"On the Hamiltonian game (a traveling salesman problem)."

Here is how she defines the problem:

"The purpose of this note is to give a method for solving a problem related
to the traveling salesman problem. It seems worthwhile to give a description
of the original problem. One formulation is to find the shortest route for a
salesman starting from Washington, visiting all the state capitals and then
returning to Washington.

More generally, to find the shortest CLOSED CURVE containing n given points 
in the plane."

for example with the following set of cities:
0, 0
1, 0
2, 0
0, 1
1, 1
2, 1
1, 2
2, 2
which can be presented as follows:
+ + +
+ + +
  + +
the shortest path is:
 _____
|__   |
   |__|

so you should print the length of this path that is:
8.00

Write a program that will read a set of city coordinates in the form 
'%f, %f\n' from the standard input and will print the length of the shortest
possible path containing all these cities under the form '%.2f'.

Your program will not be tested with more than 11 cities.

You will find in this directory a file tsp.c containing all the boring parts of
this exercise and example files to help you test your program.

hint: in order to use sqrtf, add -lm at the end of your compilation command.

For example this should work:
$> cat square.txt
1, 1
0, 1
1, 0
0, 0
$> ./tsp < square.txt | cat -e
4.00$
