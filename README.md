This repository contains my projects developed as part of the **42 School Common Core** curriculum. 42 is an innovative, peer-to-peer learning institution with campuses worldwide, known for its unique approach to software engineering education without teachers or formal classes.

Each project is designed to strengthen fundamental programming skills, system administration knowledge, and problem-solving abilities. The curriculum emphasizes autonomous learning, collaboration, and hands-on experience with real-world programming challenges.

- **Collaborative Development**: Peer-to-peer learning and code review processes

## üìö Projects Overview

### üîπ Milestone 0: Foundations
Building the fundamental tools and understanding core concepts.

- **[libft](./MILESTONE_0/libft)** - Custom implementation of standard C library functions
  - *Skills*: C programming, memory management, Makefile, string manipulation

### üîπ Milestone 1: Core Concepts
Expanding knowledge with more complex challenges.

- **[ft_printf](./MILESTONE_1/ft_printf)** - Reimplementation of the printf function
  - *Skills*: Variadic functions, formatted output, type conversion
  
- **[get_next_line](./MILESTONE_1/gnl)** - Reading from file descriptors line by line
  - *Skills*: File I/O, static variables, buffer management

### üîπ Milestone 2: Advanced Topics
Tackling algorithmic challenges and graphical programming.

- **[push_swap](./MILESTONE_2/push_swap)** - Sorting algorithm with limited operations
  - *Skills*: Algorithm optimization, complexity analysis, stack manipulation
  
- **[pipex](./MILESTONE_2/pipex)** - Unix pipe mechanism implementation
  - *Skills*: Process management, pipes, file descriptors, fork/exec
  
- **[fract-ol](./MILESTONE_2/fractol)** - Fractal visualization program
  - *Skills*: Graphics programming, mathematical algorithms, event handling

### üîπ Milestone 3: System Programming
Deep dive into concurrent programming and shell implementation.

- **[philosophers](./MILESTONE_3/philosophers)** - Dining philosophers problem
  - *Skills*: Multithreading, mutexes, synchronization, deadlock prevention
  
- **[minishell](./MILESTONE_3/minishell)** - Simple shell implementation
  - *Skills*: Process creation, signal handling, parsing, environment variables
